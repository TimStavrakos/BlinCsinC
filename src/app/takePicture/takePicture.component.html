In order for everything to work with a raspberry pi and projector you need to make sure:

  1. Your raspberry pi and projector are on
  2. Your raspberry pi is connected to the projector via the HDMI cable
  3. You have checked your connection to the raspberry pi:
    • Open cmd window (search cmd)
    • Type in: ssh pi@192.168.X.X (replacing X's with your ip)
    • If it asks for password, you are connected
    • If it does nothing and then says connection timed out, you are not connected. Wait five minutes and try again
  4. You have the camera module hooked up to the Raspberry Pi
  5. The camera is facing towards the picture you want to take
  6. Go ahead and press the Next Page button

If you are using just the web app and your personal camera, press the Manually Take Picture button to display the image!

If you are using the raspberry pi and projector, press the "Take Picture" button to display the image!
